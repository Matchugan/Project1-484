UNIQUE ELEMENT AND ATTRIBUTES (Listed on the gallery.html page)
----------------------------------------------------------------
Unique Element #1: <video>
The <video> element is used to embed a media player for video content for playback
It's used here to display a clip.
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/video


Unique Element #2: <source>
The <source> element specifies one or more media resources for media elements like <video>.
It is used here to define the MP4 file for the video clip.
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/source

            
Unique Element #3: <track>
The <track> element is used to specify timed text tracks such as subtitles or captions.
It improves accessibilty for different users of the site.
used here to priovide subtitles to the clip.
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/track


Unique Attribute #1: kind
The 'kind' attribute defines the type of text track like whether its subtitles, caption or a description.
It is used here to specify that the track provides subtitles.
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/track#kind

            
Unique Attribute #2: srclang
The 'srclang' attribute specifies the language of the text track similar to lang i would say.
It is used here to indicate that the subtitles are in English.
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/track#srclang

            
Unique Attribute #3: label
The 'label' attribute provides a readable title for the text track.
Offers a named selection in caption list.
It is used here to name the subtitle track "English" in captions.
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/track#label
